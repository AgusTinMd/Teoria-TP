Repaso HTML - CSS - JS

HTML (HyperText Markup Language): Es la estructura. Representa los cimientos, las paredes, las puertas y los techos. En la web, HTML se encarga del "esqueleto", definiendo exactamente qué contenido hay en la página: dónde va un título principal, dónde hay un bloque de texto, dónde se ubica una imagen y dónde va un botón.

CSS (Cascading Style Sheets): Es el diseño y la estética. Siguiendo con la analogía, CSS es la pintura de las paredes, el tipo de piso, la decoración y el diseño interior. En tu código, CSS toma esa estructura rígida de HTML y le da vida visual: aplica los colores, define las tipografías, ajusta los espacios entre elementos y decide si algo va centrado o alineado a la derecha.

JavaScript (JS): Es la interactividad y el comportamiento. Son las instalaciones eléctricas y de agua; lo que hace que cuando toques un interruptor, se prenda la luz, o que al abrir una canilla salga agua. En el desarrollo web, JS es el motor lógico que permite que la página reaccione a lo que hace el usuario. Se encarga de abrir menús desplegables, validar formularios, mostrar alertas o cargar nueva información sin tener que actualizar la página entera.
